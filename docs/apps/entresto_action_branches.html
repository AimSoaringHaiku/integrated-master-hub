分岐·01 エンレスト® → コララン® β遮断薬増量困難下の心拍数高値 Level 2　推奨IIa/B-R
判断基準（トリガー）
安静時心拍数 75回/分以上
β遮断薬の増量が血圧的に困難
洞調律であること（必須確認）

前提：最大耐用量β遮断薬含むGDMT実施中
HFrEF LVEF≦35% NYHA II-IV
絶対行動（アクション）
1. 心電図で洞調律を確認
2. 安静時HR 75以上を記録
3. コランラ追加を主治医へ提案
4. 高度徐脈・AF既往がないか必ず確認
絶対禁忌
高度徐脈・洞機能不全
心房細動・未治療不整脈
妊婦・重度肝障害
納得コンセプト（ロジック）
β遮断薬は「心拍数も血圧も下げる」。血圧限界で増量できない時、心拍数だけを選択的に下げるコラランが代替手段になる。

洞結節のHCNチャネルを直接抑制→洞調律が前提条件。
収穫（Tips）
「光視症（目がチカチカ）」は特有の副作用。患者への事前説明で不安を防ぐ。

AFが出たら即中止。洞調律以外では効果なし＆リスクのみ。

目標HR：60回/分以下が治療ゴール。
スクリプト 「β遮断薬をこれ以上増やすと血圧が下がりすぎるため、心拍数だけを選択的に下げるコラランの追加を提案します。心電図で洞調律、安静時心拍数75以上を確認済みです。」
モニタリング 2〜4週後 心拍数の変化 · 光視症の有無 · 心房細動の新規発現
撤退ライン 安静時HR 50回/分未満への低下　または　心房細動の発生
分岐·02 エンレスト® → ベルイシグアト® GDMT最適化下での心不全増悪 Level 2　推奨IIa/B-R
判断基準（トリガー）
十分なGDMT下での心不全増悪
（入院・利尿薬追加等のイベント）

NT-proBNP 5,314 pg/mL以下
（超過は慎重投与）

LVEF 45%未満 · 増悪歴あり
高リスクHFrEF 増悪歴あり
絶対行動（アクション）
1. PDE5阻害薬（シルデナフィル等）併用を必ず確認
2. eGFR 15以上を確認
3. NT-proBNP値を記録
4. 2週毎漸増プロトコルを説明
致命的相互作用
PDE5阻害薬と絶対併用禁忌
（シルデナフィル・タダラフィル等）
妊婦禁忌・重度肝障害禁忌
納得コンセプト（ロジック）
「GDMT最適化後もまだ増悪する」患者は、NOシグナル伝達が枯渇した状態。ベルイシグアトはsGCを直接刺激しNO回路を増強する「別ルートの心保護」。

PDE5阻害薬と重複するため、同時使用は血圧急落の危険。
収穫（Tips）
漸増スケジュールが重要：2mg→5mg→10mgと2週毎。患者に「徐々に増やす薬」と伝えると不安が減る。

貧血進行モニタリングが必須。Hb低下に注意。

勃起不全治療薬との併用禁忌を必ず確認。見落とし最多。
スクリプト 「標準治療を最適化してもなお増悪を繰り返すため、心臓のNO回路を別ルートで強化するベルイシグアトの追加を提案します。PDE5阻害薬の併用がないこと、eGFRを確認済みです。2週ごとに増量するスケジュールになります。」
モニタリング 2週毎（増量時） 血圧低下 · 貧血進行（Hb） · 増悪イベントの再発
撤退ライン 症候性低血圧（収縮期90未満）の発現　または　重度貧血の進行
分岐·03 エンレスト® → SGLT2阻害薬 腎機能悪化・eGFR低下（<30） Level 1　推奨I/A
判断基準（トリガー）
eGFR 30未満でARNI継続困難
（G4以上はエビデンス乏しい）

SGLT2iはeGFR 20以上まで使用可

HFrEF LVEF 40%以下
糖尿病の有無は不問
腎保護最優先 DM合併不問
絶対行動（アクション）
1. eGFR値を確認（20以上が最低条件）
2. 重症ケトーシス・重度感染症を除外
3. シックデイルールを説明
4. ARNIは段階的に中止（急な中断は避ける）
絶対禁忌
重症ケトーシス（eGFR問わず）
重度感染症・脱水状態
eGFR 20未満は原則使用不可
納得コンセプト（ロジック）
ARNIはG4以上でエビデンスが乏しく「使えなくなる壁」がある。SGLT2iはその壁をeGFR20まで押し下げ、心保護と腎保護を同時に提供できる。

初期のeGFR一過性低下は「正常反応」。慌てず2〜4週で再評価。
収穫（Tips）
「eGFRが下がる薬」という誤解が多い。導入初期の一過性低下を事前説明しておくことが継続率を上げる鍵。

シックデイ（発熱・嘔吐・下痢時）は自己中止を指導。ケトアシドーシス予防。

性器感染症は女性で特に注意。
スクリプト 「腎機能が低下し、エンレストを継続することが難しい状況です。SGLT2阻害薬はeGFR20まで使用でき、心臓と腎臓を同時に守るエビデンスが最も強い薬です。最初の数週間でeGFRが少し下がることがありますが、これは正常な反応で、通常2〜4週で落ち着きます。」
モニタリング 1〜2週後 eGFRの変化（急激な持続低下のみ警戒） · 脱水症状 · 性器感染
撤退ライン eGFR 20未満への持続的低下　または　性器感染・ケトアシドーシスの発症
分岐·04 エンレスト®（等） → K吸着薬を追加してARNI維持 高カリウム血症（5.0〜6.5 mEq/L） Level 1　推奨IIa/B-R
判断基準（トリガー）
血清K 5.0 mEq/L以上
RAAS阻害薬の減量・中止を
検討されている状況

心不全（LVEF不問）
高K血症合併

発想の転換：薬を減らすのではなく
K吸着薬を追加して薬を守る
絶対行動（アクション）
1. 血清K値を確認（5.0以上が適応）
2. 腸閉塞・重度便秘を除外
3. 他剤と服用間隔をあける（吸収阻害）
4. ARNIは継続・増量の方針を確認
禁忌
腸閉塞（絶対禁忌）
重度の便秘（相対禁忌）
低K血症（K <3.5）への移行に注意
納得コンセプト（ロジック）
ARNIを減量・中止すると予後が直接悪化する。高K血症はARNI中止の理由ではなく、K吸着薬を追加して解決すべき問題。

最新GLの鉄則：「心保護薬を維持するためにK吸着薬を使う」が正しい順序。
収穫（Tips）
SZC（ジルコニウム）は服用後1時間で効果発現。緊急時の初期対応にも使える速効性。

他の薬の吸収を妨げるため、服用間隔（前後2時間あける等）の指導が重要。

便秘が出たら早めに報告を。悪化すると腸閉塞リスク。
スクリプト 「カリウムが高めですが、心臓を守るエンレストを減らすことは予後の悪化につながります。カリウムを腸で吸着する新しいお薬を追加することで、エンレストを続けながらカリウムを管理する方法を提案します。他のお薬とは2時間あけて服用してください。」
モニタリング 1〜2週後 血清カリウム値 · 便秘の有無 · ARNIの維持・増量状況
撤退ライン 血清K 3.5 mEq/L未満（低K血症）への移行　または　重度便秘・腸閉塞の兆候
