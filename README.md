クロードの成果物移動

**→ [作業者トップページはこちら](https://aimsoaringhaiku.github.io/integrated-master-hub/docs/apps/)**

## 構造

```
docs/
  apps/     ← HTMLアプリ本体（index.htmlがトップ）
  data/     ← JSONデータ
  maps/     ← SVGフロー図
  tools/    ← ツール類
  sub/      ← サブシステム
  src/      ← 元資料（TSV・MD・TXT）
```
